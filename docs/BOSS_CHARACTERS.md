## 共通仕様

### ボスAoE（範囲攻撃）システム
全ボスに共通のAoEシステム:
- **hpThreshold**: HPがこの割合以下になるとAoE攻撃が有効化
- **probability**: AoE攻撃の発動確率
- **damage**: AoE攻撃のダメージ
- **range**: AoE攻撃の範囲
- **cooldownMs**: AoE攻撃のクールダウン
- **knockback**: ノックバック距離

### アニメーションキー
全ボス共通:
- `idle`: 待機アニメーション
- `walk`: 移動アニメーション
- `attack`: 攻撃アニメーション
- `die`: 死亡アニメーション

### スプライト設定
- `flipSprite`: プレイヤー側からの表示時にスプライトを反転するか
- `scale`: ベースユニットからの拡大率

## 画像生成時の共通プロンプト要素

```
Common elements for all boss characters:
- Pixel art style, 2D side-scrolling game character
- Clear silhouette, readable at various sizes
- Consistent with garden/nature theme of the game
- Cute but powerful aesthetic (similar to games like Battle Cats)
- Animation-ready sprite design
- Distinct color palette for easy recognition
- Boss-appropriate size and presence
```
